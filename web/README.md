# Web

Next.js frontend nền tảng cho `website-dong-ho-nguyen-tri`.

## Thành phần

- App Router.
- Auth provider và protected pages.
- Sidebar/layout dùng chung.
- Theme light/dark.
- Common UI components.
- Trang login, dashboard nền tảng, users, audit logs, 403, 404.

## Chạy local

```bash
pnpm --dir web dev
```

URL: `http://localhost:3000`
